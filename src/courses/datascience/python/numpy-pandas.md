# Working with NumPy and Pandas

NumPy and Pandas are essential libraries for numerical computing and data manipulation in Python.

**Topics Covered:**
- NumPy arrays
- Pandas DataFrames
- Basic data operations

**Example:**
```python
import pandas as pd
import numpy as np

arr = np.array([1, 2, 3])
df = pd.DataFrame({"Name": ["Ada", "John"], "Age": [25, 30]})
print(df)
```
