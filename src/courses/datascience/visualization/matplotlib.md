# Introduction to Matplotlib

Matplotlib is a plotting library for creating static, animated, and interactive visualizations in Python.

**Topics Covered:**
- Basic plots (line, bar)
- Customizing plots

**Example:**
```python
import matplotlib.pyplot as plt

x = [1, 2, 3]
y = [2, 4, 6]

plt.plot(x, y)
plt.show()
```
